# Tenant-scoped pattern (use when editing API controllers/models/requests)

When adding or changing **tenant-scoped** resources (Extensions, Queues, Agents, Routes, Trunks, IVRs, Inbound routes, or future e.g. Custom apps), follow this pattern. Do **not** use it for Tenants (cluster) themselves.

## Identity vs uniqueness

- **Identity (which row):** Use `id` (KSUID). Model: `primaryKey = 'id'`. In controller update: `Model::where('id', $id)->update($dirty)` and `$model->syncOriginal()` — do not rely on `$model->save()`.
- **Uniqueness:** `pkey` is unique *per cluster*. Same pkey in different clusters is allowed.

## Model

- `primaryKey = 'id'`, `$keyType = 'string'`, `$incrementing = false`
- **MUST** use **`$fillable`** (whitelist); **do not use `$guarded`.** When converting or auditing a model, replace `$guarded` with `$fillable`. Use `$hidden` only for attributes to exclude from array/JSON.
- `resolveRouteBinding($value)`: try shortuid (exact, case-insensitive), then id, then pkey

## Controller create

**REQUIRED:** For every table with `id` and `shortuid` columns, the API **must** generate and set both on create. The client must not send them. Set `id` and `shortuid` before `$model->save()`, or include them in the array for `Model::create($attrs)` (with both in model `$fillable`).

```php
move_request_to_model($request, $model, $this->updateableColumns);
$model->cluster = cluster_identifier_to_shortuid($request->cluster); // when applicable
$model->id = generate_ksuid();
$model->shortuid = generate_shortuid();
$model->save();
```

## Controller update

```php
if ($model->isDirty()) {
    $id = $model->id;
    if ($id === null || $id === '') return Response::json(['Error' => 'Resource id is missing'], 409);
    $dirty = $model->getDirty();
    Model::where('id', $id)->update($dirty);
    $model->syncOriginal();
}
```

## Form Request (pkey)

- If **update** and **pkey unchanged**: use only `'required'` for pkey (skip unique).
- Else: `Rule::unique('table','pkey')->where('cluster',$cluster)` and on update `->ignore($model->getKey(),'id')`.
